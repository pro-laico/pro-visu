---
title: palette-reel
description: A looping colour-palette reveal video — the moving counterpart of palette.
---

`palette-reel` is the moving version of [`palette`](/docs/generators/palette). The colours
start as thin slivers showing only their name; one at a time a sliver expands into a band that
reveals its configured `details` (hex / oklch / rgb …), holds, then collapses before the next
opens — sweeping every colour and looping seamlessly. Only `colors` is required. No `url`.
Output is mp4.

```ts
{
  name: "brand-colors-reel",
  generator: "palette-reel",
  options: {
    colors: [
      { name: "Ink", hex: "#1a1714" },
      { name: "Camel", hex: "#b49a77" },
      { name: "Loden", hex: "#5c5e4c" },
    ],
    details: ["hex", "oklch"],
  },
}
```

## Options

### Colours & reveal

export const coloursAndReveal = [
  { name: 'colors', type: '{ name, hex }[]', required: true, description: 'The colours to reveal (≥1).' },
  { name: 'orientation', type: '"rows" | "columns"', default: '"rows"', description: 'Horizontal bands or full-height vertical strips.' },
  { name: 'details', type: 'FieldId[]', default: '["hex","oklch","rgb"]', description: 'Fields revealed when a colour expands (name is always shown).' },
]

<ConfigOptions items={coloursAndReveal} />

### Timing (milliseconds)

export const timing = [
  { name: 'holdMs', type: 'number', default: '2000', description: 'How long each colour stays fully open (ms).' },
  { name: 'transitionMs', type: 'number', default: '700', description: 'Crossfade length between colours (ms).' },
  { name: 'bounce', type: 'boolean', default: 'true', description: 'Ping-pong the sweep so every handoff is between neighbours (avoids a last→first pinch at the loop seam). When "false", the reel wraps directly from the last item to the first — shorter, but crossfades non-adjacent bands at the seam.' },
  { name: 'easing', type: '"linear" | "ease-in" | "ease-out" | "ease-in-out"', default: '"ease-in-out"', description: 'Easing for the crossfade ramp.' },
  { name: 'durationMs', type: 'number', default: 'derived', description: 'Override the derived clean-loop length (ms).' },
]
