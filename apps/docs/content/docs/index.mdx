**auto-showcase** turns a website into marketing assets — scroll reels, responsive
screenshots, device-framed videos, scenes, font/colour specimens, and more — from a
single `showcase.config.ts`.

> This documentation site is a starting scaffold. Full content (generators, the
> `scroll-reel` deep dive, CLI, config) will be ported from the library README.

## Quick start

```bash
pnpm add -D auto-showcase
npx showcase init
npx showcase generate
```
